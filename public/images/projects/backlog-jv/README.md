# Backlog JV images

Recommended formats for V1:

- `.webp`
- max width: 1600px
- descriptive filenames, for example:
  - `dashboard.webp`
  - `filters.webp`
  - `game-detail.webp`
  - `status-board.webp`
  - `video-poster.webp`

Files in this folder are served from `/images/projects/backlog-jv/...`.
